{% trans "Gratis hosting for libre software" %}

{% blocktrans %}If you are interested in using Weblate for your libre software, you can ask to have your translations hosted on hosted.weblate.org. This allows use of Weblate without installing and maintaining it.{% endblocktrans %}

{% trans "Conditions for gratis for libre software hosting" %}

  • {% trans 'Translated content has to be released under libre license.' %}
  • {% trans "The source code has to be libre licensed, and publicly available in a supported version control system." %}
  • {% trans "There is no guarantee for service availability or quality." %}

{% trans "Asking for hosting" %}

{% blocktrans with "https://hosted.weblate.org/hosting/" as url %}Please use contact form on Hosted Weblate to request hosting. The following info is needed:{% endblocktrans %}

  • {% trans "Info about your project." %}
  • {% trans "Project website." %}
  • {% trans "URL of repository with translations." %}
  • {% trans "Path to translations within the repository." %}

{% trans "User management" %}

{% blocktrans %}The gratis for libre software plan does not allow changing project visibility, nor limiting which users can contribute to the translation.{% endblocktrans %}

{% blocktrans %}This limitation can be lifted by purchasing the "Basic" hosting plan - you have the same limits, but get extended user management capabilities.{% endblocktrans %}

{% trans "Translation workflow" %}

{% blocktrans %}All translated repositories are available using the HTTPS protocol, with similar naming as on the website - the weblate/master translation is published as https://hosted.weblate.org/git/weblate/master/. You can figure out the URL from the component info tab.{% endblocktrans %}

{% blocktrans with "https://docs.weblate.org/en/latest/admin/continuous.html#github-setup" as GitHub and "https://docs.weblate.org/en/latest/admin/continuous.html#bitbucket-setup" as Bitbucket and "https://docs.weblate.org/en/latest/admin/continuous.html#gitlab-setup" as GitLab %}Weblate can automatically receive changes from major code hosting services, see our documentation for setup on GitHub, Bitbucket or GitLab.{% endblocktrans %}

{% blocktrans with "https://docs.weblate.org/en/latest/admin/continuous.html#hosted-push" as docs %}If you want Weblate to automatically push changes as well, add the Weblate push user to collaborators, and let us know you want to use this option. See documentation for more detailed instructions.{% endblocktrans %}
